Introduction to Python & Data Science

We will be opening applications for the Fall 2023 cohort soon!










Course Info

Duration: 6 September 2023 - 13 December 2023, 15 week session

Timings: 5:45PM - 7:00PM (PST) on Wednesdays

The Introduction to Python course covers the basic core data structures built into python. We will go through the syntax, the different available functions, and when to utilize each data structure. The end goal is for each student to build a project they are passionate about using the concepts learned in class. Note: We will avoid somewhat obscure and underused data types, and instead focus on the ones which are most crucial for projects at a beginner level.
(recorded classes will be made available for students who have existing time conflicts that are shared with the instructors)

Prerequisites: None
