Course Info

Duration: 19 June 2023 - 13 July 2023, 3.5 week session

Timings: 11:30AM - 12:45PM (PST)

The data structures and algorithms course covers the fundamental concepts and techniques needed to develop efficient and reliable computer programs. Students will learn about data structures such as arrays, linked lists, stacks, and queues, and algorithms like sorting and searching. The course focuses on developing practical skills in programming using Java and applying these skills to solve real-world problems. By the end of the course, students will have a solid understanding of data structures and algorithms and be equipped to implement them in their own programming projects.
(recorded classes will be made available for students who have existing time conflicts that are shared with the instructors)

Prerequisites: Proficiency in Java or Python is Required
